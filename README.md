# Kotlin_DrawingApp

`그림 그리기` 안드로이드 앱 프로젝트 입니다. <br/><br/>
 `2022.03`월  `Android Studio`에서 `Kotlin` 을 사용해 구현했습니다.
 
이 프로젝트를 통해 이루고자 한 목표는 `파일 저장과 파일 불러오기` 였습니다. 
프로젝트 구현 과정 동안 파일처리에 대해 고민하며 코드를 작성했습니다.
